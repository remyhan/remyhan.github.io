SQL Server的安装

  • DB
  • SQL Server

本文主要介绍如何在Windows 7上安装SQL Server 2008 R2 Express。

一、文件准备

本文SQL Server选择的版本是SQL Server 2008 R2 Express，操作系统为Windows 7 （64位）。
首先请下载如下两个安装文件：

第一个文件为数据库的服务端，第二个为数据库的管理端。

二、安装SQL Server服务端

双击第一个可执行文件cn_sql_server_2008_r2_express_x64.exe，系统将会自动解压安装所需的文件

解压完成之后出现SQL Server安装中心：

选择“全新安装或向现有安装添加功能”

勾选“我接受许可条款”，并点击下一步

保证实例功能、共享功能是勾选的状态，点击下一步

如果不想安装在C盘可考虑更换安装目录，建议保持默认，点击下一步

点击下一步

选择混合模式，并且为管理员设置密码，要输入两次。点击下一步

点击下一步，程序开始安装：

如果出现以下提示则安装完成，点击关闭。

服务端安装完成，现在SQL Server安装中心界面可以关闭。

三、安装SQL Server管理端

现在双击第二个文件cn_sql_server_2008_r2_management_studio_express_x64.exe，会出现类似的SQL Server安装中心界面：

同样地，点击“全新安装或向现有安装添加功能”：

点击下一步

接受许可，点击下一步：

下一步

下一步，安装程序正式开始，等待安装结束：

四、登录到SQL Server

安装完成之后，从开始-Microsoft SQL Server 2008 R2 找到SQL Server Management Studio

可以右击将快捷方式发送至桌面：

双击桌面上SQL Server Management Studio快捷方式进入SQL Server

在服务器名称中键入 .\sqlexpress

身份验证既可以选择Windows身份验证也可以选择SQL Server身份验证。

如果选择的是SQL Server身份验证，用户名为sa，密码为安装的时候设置的密码。

点击连接。如果出现以下界面，则证明连接成功：
