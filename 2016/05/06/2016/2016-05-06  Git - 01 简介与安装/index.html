Git - 01 简介与安装

  • Git

本文主要介绍如何在Linux、Mac和Windows上安装git以及完成安装之后的配置操作。

一、简介

Git是一个分布式版本控制工具。相对于集中式版本控制系统（如CSV和SVN），分布式版本控制系统最大的特点就是系统中每一台机器中都有一个完整的版本库，这样带来很多好处：

  • 工作的时候不必时刻联网，只需提交的接入系统即可；
  • 数据的安全性更高。由于每台机器都有版本库，其中一台机器发生故障不会影响其他机器的工作。
